# apt-redesign-dummy

Noblesse / ARVED Living click-dummy — static redesign prototype of the apartment-living site.

Pure static HTML/CSS/JS (no build step). Open `index.html` locally, or view the hosted version via GitHub Pages.

## Structure
- `index.html` — landing page
- `apartments.html`, `apartment-*.html` — apartment listing + detail pages
- `heritage.html`, `event-salon.html`, `lubeck.html` — content pages
- `arved.css`, `arved.js` — shared styles + scripts
- `assets/` — images
